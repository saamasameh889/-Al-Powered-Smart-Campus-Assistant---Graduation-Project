# -Al-Powered-Smart-Campus-Assistant---Graduation-Project

🌟 Overview
AI-Powered Smart Campus Assistant is an intelligent digital assistant designed to enhance the student experience at Zewail University. It combines multiple AI technologies to provide personalized academic guidance, policy information, indoor navigation, and risk prediction in a unified platform.


🎯 Key Features

🤔 Intelligent Q&A System

Policy & Academic Information: Answer questions about university policies, courses, and regulations using RAG (Retrieval-Augmented Generation)
Context-Aware Responses: Understands natural language queries and provides accurate, cited answers

📚 Personalized Academic Planning

Course Recommendations: Suggests courses based on academic history, preferences, and success probability
Smart Scheduling: Generates semester schedules considering prerequisites, course availability, and academic rules
Retake Prioritization: Automatically prioritizes failed courses in schedule planning
⚠️ Early Warning System

Risk Prediction: Identifies at-risk students using ML models 
Proactive Alerts: Provides early warnings and intervention suggestions

🧭 Indoor Navigation
Building-to-Room Navigation: Guides students to classrooms, offices, and facilities

🤖 Multi-Agent Intelligence

Specialized AI Agents: Dedicated agents for retrieval, navigation, policy, recommendations, and final reasoning
Coordinated Workflows: Intelligent routing of queries to appropriate specialists
